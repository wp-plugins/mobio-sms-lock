Плъгина позволява да създавате платени статиите (постове), достъпни след заплащането чрез SMS към системата на Mobio.bg.
== Description ==

Locks posts and requires sms payment for unlocking it. This way you can have post with paid access

== Installation ==

1. IMPORTANT! If you are upgrading from a previous version first deactivate the plugin, then delete the plugin folder from your server.
2. Upload the plugin folder to your /wp-content/plugins/ folder. 
3. Go to the **Plugins** page and activate the plugin.
